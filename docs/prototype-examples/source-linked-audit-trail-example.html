Source-linked audit trail

Every figure in a report, business case or AI output must trace back to a source export, a documented transformation, a named reviewer and an immutable lineage hash.

The standard

A source-linked audit trail has four required parts:

  1. Source — the classified export ID the value came from.
  2. Transformation — the calculation, filter or aggregation applied.
  3. Reviewer — the named person who validated the figure.
  4. Lineage hash — a fingerprint of source + transformation + reviewer + timestamp so the chain cannot be silently altered.
