Power BI build pack example

Step-by-step build instructions for the BICC pilot.

Build steps

  1. Load classified sales invoice export into Power BI Desktop.
  2. Create net sales measure: gross sales − discounts − credit notes.
  3. Build channel/brand decomposition.
  4. Add source lineage page with export ID and refresh timestamp.
  5. Publish to BICC workspace and capture acceptance evidence.

This is a condensed example. The full build pack covers data modelling, DAX, refresh, and governance.
