Consolidated export-layer design

  1. Create a controlled export drop folder for source files.
  2. Register each file in the export inventory with classification and lineage ID.
  3. Transform using documented Power Query / Excel / Python-free formula steps available within M365 E3 workflow.
  4. Produce a single reconciled output workbook/table.
  5. Capture reconciliation checks and reviewer sign-off in the audit trail.

Validation checklist

  • Source files match inventory and naming standard.
  • Required columns are present and data types valid.
  • Totals reconcile to legacy report within agreed tolerance.
  • Manual adjustments are explicit rows with owner/reason/date.
  • Output has lineage IDs for every source and transformation.
  • Business users confirm the output matches the old report before replacement.
